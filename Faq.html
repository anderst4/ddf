Frequently Asked Questions


What do I need to build?

Install J2SE 7 SDK The build is also compatible with JDK 6.0 Update 29 (or later). Make sure that your JAVA_HOME environment variable is set to the newly installed JDK location, and that your PATH includes %JAVA_HOME%\bin (Windows) or $JAVA_HOME$/bin (*NIX).

Install Maven 3.0.3 (or later) Make sure that your PATH includes the MVN_HOME/bin directory.

Set the MAVEN_OPTS variable with the appropriate memory settings

*NIX

export MAVEN_OPTS='-Xmx512m -XX:MaxPermSize=128m'

Windows

set MAVEN_OPTS='-Xmx512m -XX:MaxPermSize=128m'

How to build?

git clone git://github.com/codice/ddf.git

Change to the top level directory of DDF source distribution.

mvn install

This will compile DDF and run all of the tests in the DDF source distribution. It usually takes some time for maven to download required dependencies in the first build. The distribution will be available under "distribution/ddf-standard/target" directory.

How to Run?

Simply Unzip the distribution. Run the executable at /bin/ddf.bat or /bin/ddf

Where do I find additional information or documentation?
